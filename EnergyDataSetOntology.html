Energy Data Set Ontology
Description

This Energy Data set Ontology helps to define data sets in Energy Data Spaces. It adds: - the technical context as the property, the schedule, and the quality associated to an energy data set - the exchange context as the participants of the data exchange. - the evaluation point attached to the data set exchange. Scope : Sharing data resources in complex environments like in the case of energy domain requires some contextual information together with data values to facilitate and ensure consistent interpretation of data.
