Description

Purpose: Optimizing self consuption, losses detection and gamification for eletric saving requires some knoweldge on LEC equipents. Scope: Local energy communities ontology provides OWL classes and properties to allow local energy equipments and their static parameters to be described. Description: The local energy communities ontology allows to describe the lec equipments and their properties. It includes the description of battery, transformer, bus, and PDD.
